Motivation

  • Distributing native software is hard
  • Time on mobile device
    • 13% Mobile Web VS 87% Apps*
  • 80% of time spent in users' top 3 apps
  • Monthly Unique Visitors
    • 3.3 Apps VS 8.9 Mobile Web
  • Acquisition Cost
    • "Selio": 4 Eur on Android App VS 0.35 Eur on Mobile Webb

What's missing?

  • Reliable performance
    • fast loading
    • working offline
    • Lie-fi
  • Fast
    • smooth UI & animations
    • jank-free scrolling
    • seamless navigation
  • Engaging
    • Home Screen
    • Push Notifications
